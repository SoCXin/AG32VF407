### 关键特性

* 248 MHz RISC-V Core (RV32IMAFC)
* 128KB SRAM + 1MB Flash
* 3x12-bit 1Msps ADC
* 10-bit 1Msps DAC
* 2K逻辑单元CPLD

### 开发环境

1：MCU部分使用VScode搭建开发环境（我们提供所有安装包）；
2：CPLD/FPGA使用Quartus开发（兼容Altera，代码可以直接参考Quartus所编写的Verilog语言），开发好通过Supra工具综合转换。
